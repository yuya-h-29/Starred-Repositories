# Swiftar

SwiftarはSwiftで書かれたStarの多い順のリポジトリを閲覧できるプロトタイプアプリです。

画面に表示されたプロジェクトをタップすると、選択したレポジトリを見ることができます。


## デモ





## インストール

こちらのアプリはプロトタイプのため、App Storeにはアップロードされておりません。以下のコマンドを使用するか、handslab-project.xcworkspaceを開いてください。

```bash
pod install
```

## 使用言語とライブラリ

- Swift
- SwiftyJSON
- Alamofire
- Kingfisher
